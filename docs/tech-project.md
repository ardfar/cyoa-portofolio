# Belidigi

- Description: A simple website to perform digital transactions such as inter-bank transfers, mobile recharge, and buying software licenses.
- Technologies: PHP, Laravel, React.js, Docker, MySQL, REST API
- Link: https://github.com/ardfar/belidigi

# SafeRec

- Description: A platform to check credibility of bank accounts, e-wallets, and QR payments.
- Technologies: PHP, Laravel, React.js, Docker, MySQL, REST API
- Link: https://github.com/ardfar/saferec

# CheckIt Js

- Description: A plain ES6 JavaScript library that provides Laravel-like input validation and configurable error display (default floating card or SweetAlert).
- Technologies: JavaScript, ES6+, DOM Manipulation, SweetAlert2
- Link: https://github.com/ardfar/CheckIt-JS

# Intelligent Photo Album

- Description: A photo album web app enhanced with “vision” features (image classification/search).
- Technologies: PHP, Laravel, React.js, Docker, MySQL, REST API, PyTorch, OpenCV, YOLO
- Link: https://github.com/ardfar/intelligent-photo-album

# Indonesia Stamp Detection

- Description: A project to recognize Indonesian stamp collections using Python and YOLOv5. Dataset annotated (via Roboflow).
- Technologies: Python, YOLOv5, PyTorch, torchvision, OpenCV, NumPy, Matplotlib, Roboflow
- link: https://github.com/ardfar/indonesia-stamp-detection

# Indonesian Law QA

- Description: A question-answering system serving answers to Indonesian law–related questions. Built on a BERT-based QA architecture.
- Technologies: Python + Flask (web app), PyTorch (torch), BERT-based models, numpy, pandas.
- Link: https://github.com/ardfar/indonesian-law-qa
